    Les affrontements #

    Ne vous attendez pas à un schéma Attaque > Défense > Points de dégât > au suivant.

    En bref

    Un Affrontement :

    • Est non-linéaire
    • Possède toujours plusieurs issues
    • Chacune apporte son lot de PX

    4 valeurs clefs gérées avec des points :

    • Les PV : votre vitalité / combativité
    • L’Endurance : votre énergie physique (souffle)
    • La Volonté : votre force mentale
    • L’Initiative : l’ordre de prise de parole des acteurs

    3 composantes clefs gérées avec des cartes :

    • Les Blessures : altèrent votre état physique
    • Les Perturbations : altèrent votre état mental
    • La Défense : vous protège
